# Todo-App

## How to start project

Create the virtual environment:

```bash
./create-venv.sh
```

Start:

```
./start.sh
```

Базовый функционал:

* регистрация/авторизация;

* работа со списками задач:

    * создание списка;
    * просмотр списков;
    * просмотр списка;
    * редактирование списка;
    * удаление списка;

* работа с задачи:

    * создание задачи;
    * просмотр задачи;
    * редактирование задачи;
    * удаление задач.
